UseAreaStaticFiles(IApplicationBuilder, IWebHostEnvironment?)

Add PhysicalFileProvider for every area that has a wwwroot folder.

To access files in the wwwroot folder of an area (for example in the css subfolder), use the following URL (both Areas and wwwroot are not specified in the URL): https://.../AreaName/css/...

Code based on: https://stackoverflow.com/a/56919679/968451 

public static void UseAreaStaticFiles(this IApplicationBuilder app, IWebHostEnvironment? environment = null)
