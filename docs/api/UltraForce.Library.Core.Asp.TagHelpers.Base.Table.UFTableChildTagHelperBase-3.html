This base class can be used for tag helpers that are placed with a UFTableTagHelperBase. The class defines protected methods to get a reference to the table, row and cell instance.

public class UFTableChildTagHelperBase<TTable, TRow, TCell> : TagHelper, ITagHelper, ITagHelperComponent where TTable : UFTableTagHelperBase where TRow : class where TCell : class
