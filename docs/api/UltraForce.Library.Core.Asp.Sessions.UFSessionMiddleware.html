UFSessionMiddleware is UltraForce.Library.NetStandard.Storage.UFKeyedStorage implementation mapping to Session.

Use Instance to access the singleton instance.

To use UFSessionMiddleware, import UFWebExtensions in Startup.cs or Program.cs and register it at IApplicationBuilder as middleware: `builder.UseUFSession();`

UseUFSession() will make a call to UseSession().
