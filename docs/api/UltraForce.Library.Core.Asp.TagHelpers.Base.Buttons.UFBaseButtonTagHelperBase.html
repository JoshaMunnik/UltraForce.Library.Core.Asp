Renders a button or link using a button styling. This class is the base class for all button tag helpers.

When rendering a button the default type is `button`; use the Submit property to change it to submit.

Renders: 

<{a|button|div} class="{GetButtonClasses()}" {href} {disabled} {target}>
  {GetBeforeCaptionHtml()}
  <span class="{GetButtonCaptionClasses()}">{children}</span>
  {GetAfterCaptionHtml()}
</{a|button|div}>
Part of the code is based on the AnchorTagHelper implemenation.
