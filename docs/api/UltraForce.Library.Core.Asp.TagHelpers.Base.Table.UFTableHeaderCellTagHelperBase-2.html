Creates a table header cell. The cell can be self-closing or can contain content with a separate closing tag.

The generated th element always uses an opening and closing tag.

When MinSize, MaxSize or Size has been set and the table is not using grid styling a style attribute is added to the td element.

Rendered html: 

<th class="{GetTableCellClasses()}" [style="min-width: {MinSize}; max-width: {MaxSize}; width: {Size}; box-sizing: content-box"]>{children}</th>

Rendered html for buttons (a button is used so that flex or grid styling can be used): 

<td class="{GetTableCellClasses()}" [style="width: {MinSize}; max-width: {MaxSize}; width: {Size}; box-sizing: content-box"]>
  <button class="{GetTableHeaderButtonClasses()} >
    {children}
  </button>
</td>
