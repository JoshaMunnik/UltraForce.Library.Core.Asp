Renders a button or link using a button styling. When rendering a button the default type is button; use the Submit property to change it to submit.

If the Type is set to Auto the class will update it to one of the type values depending on the state of the button.

Setting the Type to Disabled will style the button as disabled; however so long Disabled is not set to true, the button is still clickable.

Renders: 

<{a|button|div} class="{GetButtonCssClasses()}" {href} {disabled} {target}>
  {GetButtonIconHtml()}
  <span class="{GetButtonCaptionCssClasses()}">{children}</span>
</{a|button|div}>
Part of the code is based on the AnchorTagHelper implemenation.
