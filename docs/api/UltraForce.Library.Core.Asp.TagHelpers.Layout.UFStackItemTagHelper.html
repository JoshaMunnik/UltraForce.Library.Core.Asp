This tag should be used with UFStackTagHelper, each child within the stack should be wrapped with this tag.

When using this tag helper, make sure to include the stylesheet in the html: 

<link rel="stylesheet" href="/_content/UltraForce.Library.Core.Asp/css/uf-styles.css"/>

Renders: 

<div class="{GetStackItemClasses()}">
  {children}
</div>
[HtmlTargetElement("uf-stack-item", TagStructure = TagStructure.NormalOrSelfClosing)]
