For

An expression to be evaluated against the current model.

Date values are formatted using mysql format (so there is no confusion on month and day positions): "yyyy-mm-dd hh:mm:ss"

If the type of this value is a `bool`, an attribute `data-uf-sort-value` is set with either `0` or `1`.

The value is used as content for the item.
