SetContentToValueAsync(TagHelperOutput, ModelExpression, ViewContext)

Sets the content to the value of the ModelExpression. It handles date and bool types. Date types are formatted to "yyyy-MM-dd HH:mm:ss". Bool types are shown as a checked or unchecked checkbox.

If the model expression uses the EmailAddressAttribute the value gets rendered using a mailto: link.
