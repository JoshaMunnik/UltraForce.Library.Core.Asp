UFClickableTagHelperBase can be used as base class for elements that can be clicked upon and will jump to some other page.

It is a subclass of AnchorTagHelper adding some additional properties.

Subclasses can call ProcessHref(TagHelperOutput) to determine the href value and process the additional properties.

Properties

Call

This property can be used as a shorthand combining Controller, Action and RouteValues values. It 'calls' a controller action by building a href from the value.

Use the following format: "[Controller.]Action[(RouteValue1,RouteValue2,...)]"

Values between square brackets are optional.

If the controller is not set the current controller is used.

The text separating parameters is set by the CallParameterSeparator; the default value is a comma.

If the action does not expect any parameters, do not use the round brackets! Else the code assumes there is one parameter that has an empty string as value. 

[HtmlAttributeName("call")]
public string? Call { get; set; }
